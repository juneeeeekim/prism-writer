# PRISM Writer 사용자 매뉴얼 (v2.1)

> **최종 업데이트**: 2026-01-01

PRISM Writer는 최신 생성형 AI 기술(RAG)을 활용하여 작가와 콘텐츠 크리에이터의 글쓰기를 돕는 지능형 어시스턴트입니다.
단순한 문장 생성을 넘어, **신뢰할 수 있는 근거(Reference)**를 기반으로 글을 작성하고 검증할 수 있습니다.

## 1. 서비스 소개

PRISM Writer는 **"Fact-Grounded Writing"**을 지향합니다. 사용자가 업로드한 자료를 바탕으로 글을 작성하며, AI가 생성한 내용의 출처를 명확히 제시합니다.

### 주요 특징

| 기능 | 설명 |
|:-----|:-----|
| **프로젝트 관리** | 주제별로 문서와 채팅을 독립 관리 |
| **RAG 기반 답변** | 업로드한 문서를 검색하여 답변 생성 |
| **글쓰기 평가** | 템플릿 기반 품질 평가 시스템 |
| **휴지통** | 실수로 삭제해도 30일 내 복구 가능 |

## 2. 시작하기

### 2.1 로그인

1. PRISM Writer 웹사이트 접속

3. 대시보드로 자동 이동

### 2.2 첫 화면 구성

| 영역 | 설명 |
|:-----|:-----|
| 상단 네비게이션 | 로고, 프로필 메뉴 |
| 메인 영역 | 프로젝트 목록 |
| 하단 링크 | 휴지통 바로가기 |

## 3. 대시보드

대시보드는 모든 프로젝트를 한눈에 볼 수 있는 메인 화면입니다.

### 3.1 프로젝트 카드

각 프로젝트는 카드 형태로 표시됩니다:
- **아이콘**: 프로젝트 대표 이모지
- **이름**: 프로젝트 제목
- **설명**: 간략한 설명 (선택)
- **날짜**: 마지막 수정일

### 3.2 프로젝트 생성

1. **"+ 새 프로젝트"** 버튼 클릭
2. 프로젝트 이름 입력
3. 설명 입력 (선택)
4. 아이콘 선택 (선택)
5. **생성** 버튼 클릭

---

## 4. 프로젝트 관리

### 4.1 프로젝트 진입

프로젝트 카드를 클릭하면 해당 프로젝트의 작업 공간으로 이동합니다.

### 4.2 프로젝트 구성 요소

| 탭 | 기능 |
|:---|:-----|
| **문서** | 참고자료 업로드 및 관리 |
| **채팅** | AI와 대화 |
| **에디터** | 글쓰기 및 평가 |

### 4.3 프로젝트 삭제

1. 프로젝트 카드의 **휴지통 아이콘** 클릭
2. 확인 모달에서 **"휴지통으로 이동"** 클릭
3. 프로젝트가 휴지통으로 이동 (30일 보관)

## 5. 문서 업로드

### 5.1 지원 파일 형식

| 형식 | 확장자 |
|:-----|:------|
| PDF | `.pdf` |
| 텍스트 | `.txt` |
| 마크다운 | `.md` |

### 5.2 업로드 방법

1. 프로젝트 진입 후 **"문서"** 탭 선택
2. **드래그 앤 드롭** 또는 **파일 선택** 버튼 클릭
3. 파일 선택
4. 자동 업로드 및 벡터화 진행

### 5.3 처리 상태

| 상태 | 설명 |
|:-----|:-----|
| 업로드 중 | 파일 전송 중 |
| 처리 중 | 텍스트 추출 및 벡터화 |
| 완료 | AI 검색 가능 |
| 오류 | 처리 실패 (재시도 필요) |

## 6. AI 채팅

### 6.1 채팅 시작

1. 프로젝트 진입 후 **"채팅"** 탭 선택
2. 하단 입력창에 질문 입력
3. **Enter** 키 또는 전송 버튼 클릭

### 6.2 RAG 기반 답변

AI는 업로드된 문서를 검색하여 답변을 생성합니다:
- 관련 문서가 있으면 해당 내용을 참조하여 답변
- 참조된 문서는 답변에 표시

### 6.3 활용 예시

```
"이 프로젝트의 주요 목표가 무엇인가요?"
"마케팅 전략에 대해 정리해주세요."
"이 문서에서 핵심 키워드를 추출해주세요."
```

## 7. 에디터

### 7.1 Dual Pane 구조

| 패널 | 역할 |
|:-----|:-----|
| **왼쪽 (에디터)** | 마크다운 글쓰기 |
| **오른쪽 (어시스턴트)** | AI 도우미 기능 |

### 7.2 마크다운 지원

에디터는 마크다운 문법을 지원합니다:
- 제목: `# H1`, `## H2`, `### H3`
- 강조: `**볼드**`, `*이탤릭*`
- 목록: `- 항목`, `1. 번호`
- 인용: `> 인용문`

### 7.3 AI 어시스턴트 기능

| 탭 | 기능 |
|:---|:-----|
| **목차 제안** | 주제 기반 목차 자동 생성 |
| **참고자료** | 벡터 검색으로 관련 문서 찾기 |
| **채팅** | 글쓰기 도움 대화 |

### 7.4 글 평가

1. 글 작성 후 **"평가"** 버튼 클릭
2. AI가 여러 기준으로 글 품질 평가
3. 점수와 개선 제안 확인

## 8. 휴지통

### 8.1 휴지통 접근

대시보드 하단의 **"휴지통"** 링크 클릭

### 8.2 휴지통 기능

| 기능 | 설명 |
|:-----|:-----|
| **복구** | 프로젝트를 다시 활성화 |
| **영구 삭제** | 즉시 완전 삭제 (복구 불가) |
| **자동 삭제** | 30일 후 자동 영구 삭제 |

### 8.3 복구 방법

1. 휴지통에서 복구할 프로젝트 찾기
2. **"복구"** 버튼 클릭
3. 대시보드에서 프로젝트 확인

### 8.4 남은 기간

각 프로젝트에 **"N일 후 영구 삭제"** 표시
