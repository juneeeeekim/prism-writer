## 시작하기

### PRISM Writer란?

PRISM Writer는 **RAG(Retrieval-Augmented Generation)** 기술을 활용한 지능형 글쓰기 도우미입니다.  
여러분의 문서를 분석하여 글의 구조와 내용을 잡아주는 것이 핵심 기능입니다.

### 첫 화면

1. 웹 브라우저에서 **http://localhost:3000** 에 접속합니다.
2. 홈페이지에서 **"🚀 에디터 시작하기"** 버튼을 클릭합니다.
3. Dual Pane 에디터 화면으로 이동합니다.

## Dual Pane 에디터

### 화면 구성

| 영역                 | 위치   | 기능                      |
| :------------------- | :----- | :------------------------ |
| **에디터 패널**      | 왼쪽   | 마크다운 글쓰기           |
| **어시스턴트 패널**  | 오른쪽 | AI 도우미 기능            |
| **분할선 (Divider)** | 중앙   | 드래그하여 패널 크기 조절 |

### 패널 크기 조절

- **마우스:** 중앙 분할선을 좌우로 드래그
- **키보드:** 분할선에 포커스 후 `←` / `→` 화살표 키
- **범위:** 20% ~ 80%

## 목차 생성하기

### 목차 생성 순서

1. 오른쪽 어시스턴트 패널에서 **"🗂️ 목차 제안"** 탭 선택
2. **"📝 글의 주제"** 입력란에 작성할 글의 주제 입력
   - 예: "AI 시대의 글쓰기 방법론"
3. **"🗂️ 목차 생성"** 버튼 클릭
4. AI가 생성한 목차 확인
5. **"✅ 에디터에 삽입"** 버튼 클릭

### 생성되는 목차 예시

```
# 서론
## 배경 및 목적

# 본론
## AI 시대의 글쓰기 방법론의 핵심 개념
## 주요 방법론

# 결론
## 요약 및 제언
```

## 참고자료 활용

### 참고자료 검색

1. **"📚 참고자료"** 탭 선택
2. 검색창에 찾고 싶은 내용 입력
3. `Enter` 키 또는 🔍 버튼 클릭
4. 관련 문서 청크 목록 확인

### 참고자료 삽입

1. 원하는 참고자료 카드의 **"✍️ 삽입"** 버튼 클릭
2. 에디터에 인용 형식으로 자동 삽입됨

```markdown
> RAG는 검색과 생성을 결합한 AI 기술입니다.
>
> — _AI 기술 개요.pdf (p.12)_
```

### 참고자료 카드 정보

- **유사도:** 검색어와의 관련성 (% 표시)
- **매우 관련:** 90% 이상
- **관련:** 80% ~ 89%
- **참고:** 80% 미만

## AI 채팅

### 채팅 사용법

1. **"💬 AI 채팅"** 탭 선택
2. 하단 입력창에 질문 입력
3. `Enter` 키 또는 📤 버튼으로 전송
4. AI 응답 확인

### 활용 예시

- "이 주제에 대해 어떤 구조로 글을 쓰면 좋을까요?"
- "서론에 어떤 내용을 포함해야 할까요?"
- "이 문장을 좀 더 자연스럽게 수정해주세요."

## 키보드 단축키

| 단축키        | 기능                         |
| :------------ | :--------------------------- |
| `Tab`         | 다음 요소로 이동             |
| `Shift + Tab` | 이전 요소로 이동             |
| `←` / `→`     | 분할선 크기 조절 (포커스 시) |
| `Enter`       | 목록에서 항목 선택 / 전송    |
| `Esc`         | 모달 닫기                    |

## 자주 묻는 질문

### Q: 목차가 생성되지 않아요

**A:** 다음을 확인해주세요:

- 주제 입력란이 비어있지 않은지
- 백엔드 서버가 실행 중인지 (http://localhost:8000/health 확인)
- 네트워크 연결 상태

### Q: 참고자료 검색 결과가 없어요

**A:** 현재 데모 버전에서는 샘플 데이터만 표시됩니다.  
실제 문서를 업로드하는 기능은 향후 업데이트 예정입니다.

### Q: 글을 저장하려면 어떻게 해야 하나요?

**A:** 상단 툴바의 **"💾 저장"** 버튼을 클릭하세요.  
(Supabase 연동 후 실제 저장 기능이 활성화됩니다)

### Q: 다크 모드를 지원하나요?

**A:** 네, 시스템 설정에 따라 자동으로 전환됩니다.
