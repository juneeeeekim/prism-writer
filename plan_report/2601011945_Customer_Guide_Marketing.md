# PRISM Writer 고객 안내 가이드

> **작성일**: 2026-01-01  
> **대상**: 신규 고객, 영업팀, 마케팅팀  
> **문서 유형**: 제품 소개 및 활용 매뉴얼

---

## 🌟 PRISM Writer란?

**PRISM Writer**는 **AI가 참고자료를 학습하여 글쓰기를 도와주는 스마트 에디터**입니다.

기존 AI 글쓰기 도구들은 "아무것도 없는 상태에서" 글을 생성합니다. 하지만 **전문적인 글**을 쓰려면 **신뢰할 수 있는 자료**가 필요합니다.

PRISM Writer는 **당신의 참고자료를 AI가 이해하고**, 그 맥락에서 글을 도와줍니다.

```
📚 참고자료 업로드 → 🤖 AI가 학습 → ✍️ 맥락 기반 글쓰기 지원
```

---

## 📋 핵심 기능 안내

### 1. 📤 참고자료 업로드 & AI 학습

**무엇을 할 수 있나요?**

- PDF, TXT, MD 파일을 업로드하면 AI가 자동으로 내용을 분석합니다
- 논문, 보고서, 계약서, 회의록 등 어떤 문서든 가능합니다

**사용 방법:**

1. 에디터 화면 우측 **"참고자료"** 탭 클릭
2. **"문서 업로드"** 버튼 클릭
3. 파일 선택 → 자동 처리 완료 대기 (1~2분)

> 💡 **팁**: 한 번에 여러 파일을 업로드할 수 있습니다!

---

### 2. 💬 AI 채팅 (맥락 기반 대화)

**무엇을 할 수 있나요?**

- 업로드한 문서를 기반으로 AI와 대화합니다
- "이 보고서에서 핵심 주장이 뭐야?" 같은 질문이 가능합니다
- AI는 **당신의 자료에서** 답을 찾아줍니다

**사용 방법:**

1. 우측 **"AI 채팅"** 탭 클릭
2. 질문 입력 후 전송
3. AI가 참고자료를 검색하여 답변 제공

**예시 질문:**

- "이 논문의 연구 방법론을 요약해줘"
- "A사와 B사의 차이점이 뭐야?"
- "이 계약서에서 주의할 조항은?"

---

### 3. 📊 글 평가 시스템

**무엇을 할 수 있나요?**

- 작성 중인 글을 AI가 분석하고 점수를 매깁니다
- **논리성, 명확성, 근거 충분성** 등을 기준으로 평가
- 개선이 필요한 부분을 구체적으로 알려줍니다

**사용 방법:**

1. 우측 **"평가"** 탭 클릭
2. **"종합 평가하기"** 버튼 클릭
3. AI가 글을 분석하고 피드백 제공

> ⚠️ **주의**: 글이 50자 이상이어야 평가가 가능합니다.

---

### 4. ✨ 드래그 앤 드롭 근거 삽입

**무엇을 할 수 있나요?**

- 참고자료에서 찾은 내용을 **드래그해서 에디터에 바로 삽입**
- 출처가 자동으로 포함됩니다

**사용 방법:**

1. "참고자료" 탭에서 검색 결과 카드를 드래그
2. 에디터 영역에 드롭
3. 인용문 형태로 자동 삽입

---

### 5. 📝 "이걸로 써줘" 버튼

**무엇을 할 수 있나요?**

- 참고자료 카드에서 **"✨ 이걸로 써줘"** 버튼 클릭
- 해당 내용을 바탕으로 AI가 초안을 작성해줍니다

**사용 방법:**

1. "참고자료" 탭에서 원하는 내용 찾기
2. 카드 하단의 **"✨ 이걸로 써줘"** 클릭
3. 자동으로 AI 채팅 탭으로 이동
4. AI가 해당 내용 기반 초안 제안

---

### 6. 📁 프로젝트 관리

**무엇을 할 수 있나요?**

- 여러 글을 **프로젝트 단위**로 관리
- 프로젝트별로 참고자료가 분리됨
- 휴지통 기능으로 실수로 삭제해도 복구 가능

**사용 방법:**

1. 상단 헤더의 프로젝트 선택기 클릭
2. 새 프로젝트 생성 또는 기존 프로젝트 선택
3. 각 프로젝트는 독립적인 참고자료 공간을 가짐

---

### 7. 💾 자동 저장 & 로컬 백업

**무엇을 할 수 있나요?**

- 글을 작성하면 **2초마다 자동 저장**
- 인터넷 연결이 끊겨도 **로컬 백업**으로 복구 가능
- **Ctrl+S** 단축키로 수동 저장도 가능

## 🎯 직업별 활용 가이드

### 📰 기자 / 콘텐츠 마케터

**이럴 때 사용하세요:**

- 보도자료, 인터뷰 녹취록을 업로드하고 기사 작성
- 경쟁사 분석 자료를 넣고 비교 기사 작성
- 통계 자료를 넣고 데이터 기반 콘텐츠 제작

**추천 워크플로우:**

```
1. 취재 자료 (인터뷰, 보도자료, 통계) 업로드
2. AI에게 "이 인터뷰에서 핵심 발언 3가지 추려줘" 질문
3. 드래그 앤 드롭으로 인용문 삽입
4. AI 평가로 기사 품질 점검
```

### 📊 컨설턴트 / 리서처

**이럴 때 사용하세요:**

- 클라이언트 자료 분석 후 보고서 작성
- 여러 리서치 논문 종합하여 인사이트 도출
- 경쟁사 분석 보고서 작성

**추천 워크플로우:**

```
1. 클라이언트 제공 자료 + 시장 조사 자료 업로드
2. "이 자료들의 공통된 트렌드가 뭐야?" 질문
3. AI 제안을 바탕으로 목차 작성
4. 각 섹션별로 "이걸로 써줘" 활용하여 초안 생성
```

### 🎓 대학원생 / 연구원

**이럴 때 사용하세요:**

- 선행연구 논문 정리 및 문헌 리뷰 작성
- 연구 제안서 초안 작성
- 학술 논문의 서론/결론 작성

**추천 워크플로우:**

```
1. 관련 논문 PDF 10~20개 업로드
2. "이 논문들의 연구 방법론 차이를 비교해줘" 질문
3. 문헌 리뷰 섹션 초안 작성
4. AI 평가로 논리적 흐름 점검
```

### ⚖️ 법무팀 / 계약 관리자

**이럴 때 사용하세요:**

- 계약서 검토 및 주요 조항 정리
- 법률 자문 내용 요약
- 분쟁 관련 문서 분석

**추천 워크플로우:**

```
1. 계약서 PDF 업로드
2. "이 계약서에서 갱신 조건과 해지 조건을 정리해줘" 질문
3. 주요 조항을 드래그해서 정리 문서에 삽입
```

## 💡 의외의 활용법 (포스트잇처럼!)

> 이 섹션은 PRISM Writer의 **원래 의도와 다른, 창의적인 사용법**입니다.

---

### 🔍 1. "나만의 검색엔진" 만들기

**원래 용도:** 글쓰기를 위한 참고자료 관리

**의외의 활용:**
회사 내부 문서, 매뉴얼, 가이드라인을 모두 업로드하면 **"회사 전용 ChatGPT"**가 됩니다!

**활용 예시:**

- 신입사원: "출장비 정산 절차가 어떻게 돼?" → 사규에서 찾아서 답변
- 영업팀: "우리 제품 A와 B의 스펙 차이?" → 제품 카탈로그에서 검색
- 개발팀: "이 API 사용법?" → 기술 문서에서 검색

> **🎯 판매 포인트:** "외부 AI에 회사 정보 넣으면 정보 유출 걱정되시죠? PRISM Writer는 **당신의 자료만** 사용합니다."

---

### 📚 2. "독서 노트" 자동화

**원래 용도:** 전문 자료 기반 글쓰기

**의외의 활용:**
읽은 책 PDF를 업로드하고, AI에게 질문하면 **개인 독서 비서**가 됩니다!

**활용 예시:**

- "이 책에서 저자가 말하는 핵심 3가지는?"
- "5장에서 나온 사례를 정리해줘"
- "이 책의 내용을 블로그 포스팅으로 써줘"

> **🎯 판매 포인트:** "책을 다 읽을 시간이 없으세요? **핵심만 뽑아드립니다.**"

---

### 🎤 3. "회의록 → 보고서" 자동 변환

**원래 용도:** 참고자료 기반 문서 작성

**의외의 활용:**
회의 녹취록을 업로드하면 **보고서나 이메일 초안**을 자동 생성!

**활용 예시:**

1. 회의 녹음 → 자동 전사 (Whisper 등 외부 도구)
2. 전사 텍스트 PRISM Writer에 업로드
3. "이 회의 내용을 임원 보고서로 작성해줘" 요청
4. 형식 갖춘 보고서 초안 완성

> **🎯 판매 포인트:** "회의 끝나고 **10분 만에** 보고서 초안이 나옵니다."

---

### 📧 4. "과거 이메일 검색 비서"

**원래 용도:** 문서 검색 및 인용

**의외의 활용:**
중요한 이메일들을 텍스트로 저장해서 업로드하면, **이메일 검색이 초간단**!

**활용 예시:**

- "작년에 A사와 나눈 협상 조건이 뭐였지?"
- "B 프로젝트 관련 메일에서 일정이 어떻게 됐었지?"

> **🎯 판매 포인트:** "Outlook 검색으로 못 찾는 정보, **AI가 찾아드립니다.**"

---

### 🧠 5. "학습 자료 정리 도우미"

**원래 용도:** 전문 글쓰기 지원

**의외의 활용:**
온라인 강의 자료, 교재 PDF를 업로드하면 **개인 튜터**가 됩니다!

**활용 예시:**

- 자격증 교재 업로드 → "시험에 자주 나오는 개념 10개 정리해줘"
- 코딩 강의 자료 업로드 → "이 개념 쉽게 설명해줘"
- 면접 준비 자료 업로드 → "예상 질문 5개 뽑아줘"

> **🎯 판매 포인트:** "**과외 선생님 없이도** 개인 맞춤 학습이 가능합니다."

## 📞 자주 묻는 질문 (FAQ)

### Q: 업로드한 자료가 외부로 유출되나요?

> **A**: 아니요. 모든 자료는 **당신의 계정 내에서만** 사용됩니다. 다른 사용자의 자료와 절대 섞이지 않습니다.

### Q: 어떤 파일 형식을 지원하나요?

> **A**: PDF, TXT, MD (마크다운) 파일을 지원합니다. DOCX는 곧 지원 예정입니다.

### Q: 한 번에 몇 개까지 업로드할 수 있나요?

> **A**: 프로젝트당 제한 없이 업로드 가능합니다. 다만, 대용량 PDF는 처리에 시간이 걸릴 수 있습니다.

### Q: 영어 문서도 지원하나요?

> **A**: 네, 영어/한국어 모두 지원합니다. AI 채팅은 한국어로 질문하시면 됩니다.

### Q: 저장한 글이 갑자기 사라지면 어떡하나요?

> **A**: 걱정 마세요! 2초마다 자동 저장되고, 인터넷 끊겨도 로컬 백업이 있습니다. 휴지통 기능도 있어서 삭제해도 복구 가능합니다.
